static SCTP_instance* sctpInstance

Whenever an external event (ULP-call, socket-event or timer-event) this variable must contain the addressed sctp instance.

Documentation

Whenever an external event (ULP-call, socket-event or timer-event) this variable must contain the addressed sctp instance. This pointer must be reset to null after the event has been handled.
