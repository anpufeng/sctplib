int sctp_setAssocStatus(unsigned int associationID, SCTP_AssociationStatus* new_status)

sctp_setAssocStatus allows for setting a number of association parameters.

Documentation

sctp_setAssocStatus allows for setting a number of association parameters. _Not_ all values that the corresponding sctp_getAssocStatus-function returns may be SET here ! Will set protocol parameters per SCTP association

Parameters:
associationID - ID of assocation.
new_status - pointer to new parameters
Returns:
-1
