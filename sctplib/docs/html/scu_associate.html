void scu_associate(unsigned short noOfOutStreams, unsigned short noOfInStreams)

Documentation

This function is called to initiate the setup an association.

The local tag and the initial TSN are randomly generated. Together with the parameters of the function, they are used to create the init-message. This data are also stored in a newly created association-record.

Parameters:
noOfOutStreams - number of send streams.
noOfInStreams - number of receive streams.
