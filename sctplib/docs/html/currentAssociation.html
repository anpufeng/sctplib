static Association* currentAssociation

Whenever an external event (ULP-call, socket-event or timer-event) this variable must contain the addressed association.

Documentation

Whenever an external event (ULP-call, socket-event or timer-event) this variable must contain the addressed association. Read functions for 'global data' read data from the association pointed to by this pointer. This pointer must be reset to null after the event has been handled.
