gboolean scr_init(SCTP_init * init)

scr_init is called by bundling when a init message is received from the peer.

Documentation

scr_init is called by bundling when a init message is received from the peer. an InitAck may be returned, alongside with a cookie chunk variable parameter. The following data are created and included in the init acknowledgement: a COOKIE parameter.
Parameters:
init - pointer to the received init-chunk (including optional parameters)
