void pm_rto_backoff(short pathID)

pm_rto_backoff is called by reliable transfer when the T3 retransmission timer expires.

Documentation

pm_rto_backoff is called by reliable transfer when the T3 retransmission timer expires. Each call of this function may double the RTO (timer back off).
Parameters:
pathID - index of the address where the timeout occurred
