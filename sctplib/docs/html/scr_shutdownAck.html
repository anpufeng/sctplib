gboolean scr_shutdownAck()

scr_shutdownAck is called by bundling when a shutdownAck chunk was received from the peer.

Documentation

scr_shutdownAck is called by bundling when a shutdownAck chunk was received from the peer. Depending on the current state of the association, COMMUNICATION LOST is signaled to the Upper Layer Protocol, and the association marked for removal.
