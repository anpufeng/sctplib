void sci_allChunksAcked()

Called by reliable transfer if all (sent !) chunks in its retransmission queue have been acked.

Documentation

Called by reliable transfer if all (sent !) chunks in its retransmission queue have been acked. This function is used to move from state SHUTDOWNPENDING to SHUTDOWNSENT (after having sent a shutdown chunk) or to move from SHUTDOWNRECEIVED to SHUTDOWNACKSENT (after having sent a shutdown-ack chunk)
