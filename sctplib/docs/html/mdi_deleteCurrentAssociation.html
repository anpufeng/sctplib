mdi_deleteCurrentAssociation deletes the current association.

Documentation

mdi_deleteCurrentAssociation deletes the current association.

The association will not be deleted at once, but is only marked for deletion. This is done in this way to allow other modules to finish their current activities. To prevent them to start new activities, the currentAssociation pointer is set to NULL.
