void scr_cookie_echo(SCTP_cookie_echo * cookie_echo)

scr_cookie_echo is called by bundling when a cookie echo chunk was received from the peer.

Documentation

scr_cookie_echo is called by bundling when a cookie echo chunk was received from the peer. The following data is retrieved from the cookie and saved for this association:
  • from the init chunk:
    • peers tag
    • peers receiver window credit
    • peers initial TSN
    • peers network address list if multihoming is used
  • local tag generated before the initAck was sent
  • my initial TSN generated before the initAck was sent
  • number of send streams I use, must be lower or equal to peers number of receive streams from init chunk
  • number of receive streams I use (can be lower than number of send streams the peer requested in the init chunk \end{itemiz}
Parameters:
cookie_echo - pointer to the received cookie echo chunk
