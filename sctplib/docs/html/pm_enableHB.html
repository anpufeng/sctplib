int pm_enableHB(short pathID, unsigned int hearbeatIntervall)

pm_enableHB is called when ULP wants to enable heartbeat.
Parameters:
pathID - index of address, where we sent the HBs to
hearbeatIntervall - time in msecs, that is to be added to the RTT, before sending HB
Returns:
error code, 0 for success, 1 for error (i.e. address index too large)
